About This Project

MoodMelody is a web app created for the INST 377 final project. It helps users generate personalized music playlists based on their mood and preferred music genre. By combining emotion analysis and real-time track data from the iTunes API, the app delivers a unique and dynamic music experience.

The app uses frontend JavaScript (with Swiper.js and TailwindCSS), backend API routes built with Node.js, and a Supabase database to store user moods. It was designed with accessibility, creativity, and user personalization in mind.

Created by a student team at the University of Maryland. Special thanks to our instructors and classmates in INST 377 for support and feedback.
